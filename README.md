# 👁️ WORLD-VIEW ™

### Unified Economic Situation Index (UESI) – 3–6 Month Forward Predictor

**WORLD-VIEW** is a browser-based economic dashboard that predicts global macroeconomic stress, commodity prices (Gold, Silver, Crude Oil), and living standard trajectories using a proprietary **Unified Economic Situation Index (UESI)**.

Built entirely in vanilla HTML/CSS/JS, it runs locally in your browser—no servers, no API keys, no installation required.

## 🧮 The Core Formula

The engine is driven by a 6-equation lead-lag system:

```

UESI = ΔGDP – ΔInflation – Ψ + 0.1 × (Gold/Silver – 60)

```

Where **Ψ (Debt-Squeeze)** = `(Household Debt / Income) × (Central Bank Rate – Inflation)`

**UESI Thresholds:**
- **> +1.5** → Total Comfortable Recovery (Expansion)
- **+0.5 to +1.5** → Recovery Initiated
- **-0.5 to +0.5** → Stagnant Transition
- **-1.5 to -0.5** → Recessionary Pressure (Downturn)
- **< -1.5** → ⚠️ Systemic Deflationary Bust

## 🌟 Features

- **📅 Historical Validation (2000–2026)** – Instantly load real macro data for major crisis years (2008 GFC, 2014 Oil Crash, 2020 COVID, 2022 Inflation Spike) to see how the formula reacts.
- **🌍 20+ Global Regions** – Track living standards (Degrading / Stable / Improving) for the USA, China, India, Germany, Japan, Brazil, Russia, Thailand, UAE, Saudi Arabia, South Africa, and more.
- **📡 Live Commodity Feeds** – One-click fetch for real-time Gold, Silver, and Oil prices. Gracefully handles partial API failures (you can manually type missing values).
- **📈 3-Month Price Forecasts** – Predicts directional movement for Gold, Silver, and Crude Oil.
- **💻 Zero Dependencies** – 100% client-side. Open `index.html` and go.

## 🚀 How to Use

1.  Clone or download this repository.
2.  Open **`index.html`** in any modern web browser (Chrome, Edge, Firefox).
3.  **Select a Year** from the dropdown (2000–2026) to load historical data.
4.  **Choose a Region** to see localized living standard estimates.
5.  Click **"Fetch Live Prices"** to load current commodity data (or manually enter values).
6.  Click **"Run 6-Month Prediction"** to calculate the UESI Score and forward forecasts.
